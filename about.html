About me

Software Developer trained in Full Stack Web Development. Passionate about coding and excited about learning and integrating new technologies into my projects. Highly proficient in developing MERN Stack applications. Also have experience developing with the .NET Core framework using C# and the Django framework using Python. Front-end experience includes developing dynamic web applications using ReactJS, Angular, TypeScipt, JavaScript, JQuery, Bootstrap, CSS, and XML/HTML. Back-end development experience includes building node express servers using MVC architecture, passport user authentication, and RESTful API development using MySQL and MongoDB/Mongoose.

I recently took on a leadership role in developing a MERN Stack fitness and social media application called SpartanFit. The SpartanFit web application allows users to create a profile, track their workout progress, search and save food recipes, and connect with other users on SpartanFit Social. My role in the project was development of the backend modeling and routing, passport user authentication, front-end design, and development of social media ReactJS components. My career in public education has allowed me to develop strong organization, collaboration, and multitasking skills, all of which enable me to be a valuable and productive member of any software development team.
