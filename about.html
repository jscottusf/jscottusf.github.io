About me

Software Developer trained in Full Stack Web Development. Passionate about coding and excited about learning and integrating new technologies into my projects. Especially strong in developing MERN Stack applications using MongoDB/Mongoose, Express, ReactJS, and Node.js. Also have some experience developing with the .Net Core framwork using C# and the Django framework using Python. Front-end experience includes developing dynamic web applications using ReactJS, JavaScript, JQuery, Bootstrap, CSS, and XML/HTML. Back-end development experience includes building node express servers using MVC architecture, passport user authentication, and RESTful API development using MySQL and MongoDB. In addition, I am highly proficient and integrating API data into front-end and back-end applications using Ajax, Axios, and express-handlebars.

I recently took on a leadership role in developing a full-stack web application called CriShipped, which functions as a resource for finding and distributing limited supplies and resources during times of crisis. Some of the features I developed include user authentication, user profiles, a discussion forum for exchanging supplies, a shopping cart for processing orders, and integration of MySQL and MongoDB databases. My career in public education has allowed me to develop strong organization, collaboration, and multitasking skills, all of which enable me to be a valuable and productive member of any software development team.
